# Full-Stack Development Journey

A comprehensive collection of production-ready starters, architectural patterns, and best practices across modern frameworks. This repository demonstrates SOLID principles and Clean Architecture implementations.

## 📐 Architecture & Principles

Detailed guidelines, code examples, and patterns are documented in [ARCHITECTURE.md](ARCHITECTURE.md).

**Key Focus Areas:**
- **SOLID Principles**: SRP, OCP, LSP, ISP, DIP
- **Clean Architecture**: Layer separation and dependency rules
- **Testing**: In-memory repository patterns and integration tests
- **Deployment**: Standardized configurations and CI/CD workflows

## 🚀 Quick Start

1. **Clone**: `git clone https://github.com/ibra-kdbra/Journey_FullStack.git`
2. **Explore**: Navigate to any project directory (e.g., `cd next-prisma-starter`).
3. **Setup**: Most projects use `npm install` or `bun install` followed by `npm run dev`.
4. **Docs**: Refer to individual project `README.md` files for specific instructions.
