## Project Introduction

This project is a imitation of the [Honor Official Website Homepage](https://www.honor.com/global/magic-os-7/) website.

## Technical Stack

* React
* Gsap
* TypeScript

## Run

Start the project

```
npm install
```

```
npm dev
```

Build the Project

```
npm build
```
